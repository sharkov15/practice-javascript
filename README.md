<h1>Практика на JavaScript</h1>
<p>Небольшое приложение для прокачки реакции, можно использовать для тренировки перед важными играми</p>
<h2>Как играть?</h2>
<p>Для начала откройте папку с файлами проекта, затем вы увидете файл index.html, запустите этот файл (он должен открытся в вашем браузере) и начинайте играть</p>
